Target Sum

View on LeetCode ↗
You are given an integer array nums and an integer target. Build an expression by placing + or − before each integer. Return the number of different expressions that evaluate to target. Strategy: DP hashmap — track reachable sums and their way-counts, expanding each number.
Solution code
def findTargetSumWays(self, nums, target):
    counter = {0: 1}
    for n in nums:
        temp = {}
        for total, count in counter.items():
            temp[total + n] = temp.get(total + n, 0) + count
            temp[total - n] = temp.get(total - n, 0) + count
        counter = temp
    return counter.get(target, 0)
