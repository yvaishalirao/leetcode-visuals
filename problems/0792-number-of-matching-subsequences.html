Number of Matching Subsequences

View on LeetCode ↗
Given a string s and an array of strings words, return the number of words[i] that is a subsequence of s. A subsequence is a string that can be derived from another by deleting some characters without changing the order of the remaining ones.
Solution code
class Solution:
    def numMatchingSubseq(self, s: str, words: List[str]) -> int:
        def is_sub(word):
            index = -1
            for ch in word:
                index = s.find(ch, index + 1)
                if index == -1:
                    return False
            return True
        c = 0
        for word in words:
            if is_sub(word):
                c += 1
        return c
