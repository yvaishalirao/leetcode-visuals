Unique Paths

LeetCode ↗

A robot at top-left of an m×n grid can only move right or down. Count unique paths to bottom-right.

Example: m=3, n=7 → 28  |  m=3, n=2 → 3

Solution code
def uniquePaths(self, m, n):
    dp = [[0] * n for _ in range(m)]
    for i in range(0, m):
        dp[i][0] = 1
    for i in range(0, n):
        dp[0][i] = 1
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i][j-1] + dp[i-1][j]
    return dp[m-1][n-1]
