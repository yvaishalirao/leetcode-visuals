Triangle

Given a triangle array, return the minimum path sum from top to bottom. At each step you may move to an adjacent number on the row below. Strategy: bottom-up DP — update each cell in-place by adding the smaller of its two children.
Solution code
def minimumTotal(self, t):
    n = len(t)
    for i in range(n-2, -1, -1):
        for j in range(0, len(t[i])):
            t[i][j] += min(t[i+1][j], t[i+1][j+1])
    return t[0][0]
