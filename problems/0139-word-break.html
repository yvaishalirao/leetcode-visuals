Word Break

View on LeetCode ↗
Given string s and dictionary wordDict, return true if s can be segmented into dictionary words. dp[i] = can s[0:i] be segmented? For each end position i, try every start j: if dp[j] is true and s[j:i] is in the dict, set dp[i] = true.
Solution code
class Solution(object):
    def wordBreak(self, s, wordDict):
        dp = [False] * (len(s) + 1)
        dp[0] = True
        for i in range(1, len(s) + 1):
            for j in range(i):
                if dp[j] and s[j:i] in wordDict:
                    dp[i] = True
                    break
        return dp[len(s)]
