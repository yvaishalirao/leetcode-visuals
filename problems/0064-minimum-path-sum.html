Minimum Path Sum

LeetCode ↗

Find a path from top-left to bottom-right of an m×n grid (moving only right or down) that minimises the sum of all numbers along the path.

Example: [[1,3,1],[1,5,1],[4,2,1]] → 7  (1→3→1→1→1)

Solution code
def minPathSum(self, grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]
    dp[0][0] = grid[0][0]
    for i in range(1, n):
        dp[0][i] = dp[0][i-1] + grid[0][i]
    for i in range(1, m):
        dp[i][0] = dp[i-1][0] + grid[i][0]
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])
    return dp[m-1][n-1]
