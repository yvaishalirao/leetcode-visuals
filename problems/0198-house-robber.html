House Robber

View on LeetCode ↗
Given an integer array nums representing the amount of money at each house, return the maximum amount you can rob tonight. You cannot rob two adjacent houses — they share an alarm system.
Solution code
class Solution(object):
    def rob(self, nums):
        n = len(nums)
        if n < 2:
            return nums[0]
        dp = [0] * n
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        for i in range(2, n):
            dp[i] = max(dp[i-2] + nums[i], dp[i-1])
        return dp[-1]
