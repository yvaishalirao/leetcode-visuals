Coin Change II

View on LeetCode ↗
Given an array of coin denominations and a total amount, return the number of distinct combinations that make up that amount. Order does not matter — [1,2] and [2,1] are the same combination. You may use each coin an unlimited number of times.
Solution code
def change(self, amount, coins):
    dp = [0] * (amount + 1)
    dp[0] = 1
    for c in coins:
        for i in range(c, amount + 1):
            if dp[i - c]:
                dp[i] += dp[i - c]
    return dp[-1]
