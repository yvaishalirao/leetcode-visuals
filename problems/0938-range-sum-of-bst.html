Range Sum of BST

View on LeetCode ↗
Given the root of a binary search tree and two integers low and high, return the sum of values of all nodes with a value in the inclusive range [low, high]. The BST property lets the recursion skip entire subtrees that can't possibly contain a value in range.
Solution code
class Solution(object):
    def rangeSumBST(self, root, low, high):
        if not root:
            return 0
        sum=0
        if root.val>low:
            sum+=self.rangeSumBST(root.left,low,high)
        if root.val>=low and root.val<=high:
            sum+=root.val
        if root.val<high:
            sum+=self.rangeSumBST(root.right,low,high)
        return sum
