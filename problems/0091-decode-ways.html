Decode Ways

View on LeetCode ↗
A message is encoded as a string of digits ("1"→A, "2"→B, …, "26"→Z). Given a string s, return the number of ways to decode it. Strategy: 1-D DP — at each position check if the last 1 digit or last 2 digits form a valid code.
Solution code
def numDecodings(self, s):
    n = len(s)
    dp = [0] * (n + 1)
    c = [str(i+1) for i in range(26)]
    if s[0] == '0':
        return 0
    dp[0] = dp[1] = 1
    for i in range(2, n + 1):
        a = dp[i-1] if s[i-1] in c else 0
        b = dp[i-2] if s[i-2:i] in c else 0
        dp[i] = a + b
    return dp[-1]
