Coin Change

View on LeetCode ↗
Given an array of coin denominations and a total amount, return the fewest number of coins needed to make up that amount. If it cannot be made up by any combination, return -1. You may use each coin an unlimited number of times.
Solution code
class Solution(object):
    def coinChange(self, coins, amount):
        dp = [float('inf')] * (amount + 1)
        dp[0] = 0
        for i in range(1, amount + 1):
            for coin in coins:
                if i - coin >= 0:
                    dp[i] = min(dp[i], 1 + dp[i - coin])
        return dp[amount] if dp[amount] != float('inf') else -1
