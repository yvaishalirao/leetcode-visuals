Count Subarrays With Majority Element I

View on LeetCode ↗
Given an integer array nums and an integer target, return the number of subarrays of nums where target is the majority element. The majority element of a subarray is the element that appears more than half the length of the subarray.
Solution code
class Solution(object):
    def countMajoritySubarrays(self, nums, target):
        n = len(nums)
        ans = 0
        for l in range(n):
            target_count = 0
            for r in range(l, n):
                if nums[r] == target:
                    target_count += 1
                length = r - l + 1
                if target_count > length // 2:
                    ans += 1
        return ans
