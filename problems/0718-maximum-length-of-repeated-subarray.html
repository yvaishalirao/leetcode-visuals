Maximum Length of Repeated Subarray

View on LeetCode ↗
Given two integer arrays nums1 and nums2, return the maximum length of a subarray that appears in both. Strategy: 2-D DP — dp[i][j] is the length of the longest common subarray ending at nums1[i-1] and nums2[j-1]; reset to 0 on mismatch, track global max.
Solution code
def findLength(self, nums1, nums2):
    m, n = len(nums1), len(nums2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    maxx = 0
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if nums1[i-1] == nums2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
                maxx = max(dp[i][j], maxx)
            else:
                dp[i][j] = 0
    return maxx
