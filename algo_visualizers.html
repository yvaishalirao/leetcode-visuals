Interleaving String

View on LeetCode ↗
Given strings s1, s2, and s3, return true if s3 is formed by an interleaving of s1 and s2. An interleaving maintains the relative order of characters from each string but can mix them in any order.
Solution code
class Solution:
    def isInterleave(self, s1, s2, s3):
        if len(s1) + len(s2) != len(s3): return False
        m, n = len(s1), len(s2)
        dp = [[False] * (n + 1) for _ in range(m + 1)]
        dp[0][0] = True
        for i in range(m + 1):
            for j in range(n + 1):
                if not dp[i][j]: continue
                if i < m and s1[i] == s3[i + j]:
                    dp[i + 1][j] = True
                if j < n and s2[j] == s3[i + j]:
                    dp[i][j + 1] = True
        return dp[m][n]

Count Subarrays With Majority Element I

View on LeetCode ↗
Given an integer array nums and an integer target, return the number of subarrays of nums where target is the majority element. The majority element of a subarray is the element that appears more than half the length of the subarray.
Solution code
class Solution(object):
    def countMajoritySubarrays(self, nums, target):
        n = len(nums)
        ans = 0
        for l in range(n):
            target_count = 0
            for r in range(l, n):
                if nums[r] == target:
                    target_count += 1
                length = r - l + 1
                if target_count > length // 2:
                    ans += 1
        return ans

Find the Maximum Number of Elements in Subset

View on LeetCode ↗
Given an integer array nums, find the maximum length of a subset that can be arranged as [x, x², x⁴, …, x^(2^k), …, x^(2^k), …, x⁴, x², x] — a palindrome-like pattern where each element is the square of the previous. The subset must have odd length.
Solution code
class Solution:
    def maximumLength(self, nums):
        from collections import Counter
        freq = Counter(nums)
        ans = 1
        if 1 in freq:
            cnt = freq[1]
            ans = max(ans, cnt if cnt % 2 else cnt - 1)
        for start in freq:
            if start == 1: continue
            cur = start
            length = 0
            while cur in freq:
                if freq[cur] >= 2:
                    length += 2
                    cur *= cur
                else:
                    length += 1
                    break
            if length % 2 == 0:
                length -= 1
            ans = max(ans, length)
        return ans

Coin Change

View on LeetCode ↗
Given an array of coin denominations and a total amount, return the fewest number of coins needed to make up that amount. If it cannot be made up by any combination, return -1. You may use each coin an unlimited number of times.
Solution code
class Solution(object):
    def coinChange(self, coins, amount):
        dp = [float('inf')] * (amount + 1)
        dp[0] = 0
        for i in range(1, amount + 1):
            for coin in coins:
                if i - coin >= 0:
                    dp[i] = min(dp[i], 1 + dp[i - coin])
        return dp[amount] if dp[amount] != float('inf') else -1

Given an array of coin denominations and a total amount, return the number of distinct combinations that make up that amount. Order does not matter — [1,2] and [2,1] are the same combination. You may use each coin an unlimited number of times.
Solution code
def change(self, amount, coins):
    dp = [0] * (amount + 1)
    dp[0] = 1
    for c in coins:
        for i in range(c, amount + 1):
            if dp[i - c]:
                dp[i] += dp[i - c]
    return dp[-1]

Number of Matching Subsequences

View on LeetCode ↗
Given a string s and an array of strings words, return the number of words[i] that is a subsequence of s. A subsequence is a string that can be derived from another by deleting some characters without changing the order of the remaining ones.
Solution code
class Solution:
    def numMatchingSubseq(self, s: str, words: List[str]) -> int:
        def is_sub(word):
            index = -1
            for ch in word:
                index = s.find(ch, index + 1)
                if index == -1:
                    return False
            return True
        c = 0
        for word in words:
            if is_sub(word):
                c += 1
        return c

Solution code
class Solution(object):
    def climbStairs(self, n):
        if n <= 2:
            return n
        a = 1
        b = 2
        for i in range(3, n + 1):
            curr = a + b
            a = b
            b = curr
        return b

House Robber

View on LeetCode ↗
Given an integer array nums representing the amount of money at each house, return the maximum amount you can rob tonight. You cannot rob two adjacent houses — they share an alarm system.
Solution code
class Solution(object):
    def rob(self, nums):
        n = len(nums)
        if n < 2:
            return nums[0]
        dp = [0] * n
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        for i in range(2, n):
            dp[i] = max(dp[i-2] + nums[i], dp[i-1])
        return dp[-1]

Jump Game

View on LeetCode ↗
You are at index 0 of integer array nums. Each element is your max jump length from that position. Return true if you can reach the last index. Strategy: scan backwards and try to walk the goal pos all the way back to index 0.
Solution code
class Solution(object):
    def canJump(self, nums):
        if nums[0] == 0 and len(nums) > 1:
            return False
        n = len(nums)
        pos = n - 1
        for i in range(n - 2, -1, -1):
            if i + nums[i] >= pos:
                pos = i
        return pos < 2

        dp = [False] * (len(s) + 1)

Jump Game II

View on LeetCode ↗
Return the minimum number of jumps to reach the last index. curr_end is the boundary of the current jump's window. Scan forward updating max_reach; the moment i hits curr_end, take a jump and extend the window to max_reach.
Solution code
class Solution:
    def jump(self, nums):
        n = len(nums)
        if n <= 1:
            return 0
        jumps = 0
        curr_end = 0
        max_reach = 0
        for i in range(n - 1):
            max_reach = max(max_reach, i + nums[i])
            if i == curr_end:
                jumps += 1
                curr_end = max_reach
                if curr_end >= n - 1:
                    break
        return jumps

Longest Common Subsequence

View on LeetCode ↗
Given two strings text1 and text2, return the length of their longest common subsequence. A subsequence is derived by deleting some characters without changing relative order. If there is no common subsequence, return 0.
Solution code
def longestCommonSubsequence(self, text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

Target Sum

View on LeetCode ↗
You are given an integer array nums and an integer target. Build an expression by placing + or − before each integer. Return the number of different expressions that evaluate to target. Strategy: DP hashmap — track reachable sums and their way-counts, expanding each number.
Solution code
def findTargetSumWays(self, nums, target):
    counter = {0: 1}
    for n in nums:
        temp = {}
        for total, count in counter.items():
            temp[total + n] = temp.get(total + n, 0) + count
            temp[total - n] = temp.get(total - n, 0) + count
        counter = temp
    return counter.get(target, 0)

View on LeetCode ↗
Given two strings word1 and word2, return the minimum number of operations to convert word1 to word2. Allowed operations: insert, delete, or replace a character. Strategy: 2-D DP — dp[i][j] = min ops for word1[0..i-1] → word2[0..j-1].
Solution code
def minDistance(self, word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        dp[i][0] = i
    for j in range(1, n + 1):
        dp[0][j] = j
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = min(dp[i-1][j-1], dp[i-1][j], dp[i][j-1]) + 1
    return dp[m][n]

def numDecodings(self, s):

        dp[i] = a + b
    return dp[-1]

Given an integer array nums, return the length of the longest strictly increasing subsequence.

Example: nums=[10,9,2,5,3,7,101,18] → 4 ([2,3,7,101])

Solution code
def lengthOfLIS(self, nums):
    n = len(nums)
    dp = [1] * n
    ans = 1
    for i in range(1, n):
        for j in range(0, i):
            if nums[i] > nums[j]:
                dp[i] = max(dp[i], dp[j] + 1)
        ans = max(dp[i], ans)
    return ans

Unique Paths

LeetCode ↗

A robot at top-left of an m×n grid can only move right or down. Count unique paths to bottom-right.

Example: m=3, n=7 → 28  |  m=3, n=2 → 3

Solution code
def uniquePaths(self, m, n):
    dp = [[0] * n for _ in range(m)]
    for i in range(0, m):
        dp[i][0] = 1
    for i in range(0, n):
        dp[0][i] = 1
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i][j-1] + dp[i-1][j]
    return dp[m-1][n-1]

Minimum Path Sum

LeetCode ↗

Find a path from top-left to bottom-right of an m×n grid (moving only right or down) that minimises the sum of all numbers along the path.

Example: [[1,3,1],[1,5,1],[4,2,1]] → 7  (1→3→1→1→1)

Solution code
def minPathSum(self, grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]
    dp[0][0] = grid[0][0]
    for i in range(1, n):
        dp[0][i] = dp[0][i-1] + grid[0][i]
    for i in range(1, m):
        dp[i][0] = dp[i-1][0] + grid[i][0]
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])
    return dp[m-1][n-1]

Unique Paths II

LeetCode ↗

Robot at (0,0) must reach (m-1,n-1), moving only right or down. Cells with 1 are obstacles. Count unique paths.

Example: [[0,0,0],[0,1,0],[0,0,0]] → 2

Solution code
def uniquePathsWithObstacles(self, obstacleGrid):
    m, n = len(obstacleGrid), len(obstacleGrid[0])
    dp = [[0] * n for _ in range(m)]
    if obstacleGrid[0][0] == 1:
        return 0
    dp[0][0] = 1
    for i in range(m):
        for j in range(n):
            if obstacleGrid[i][j] == 1:
                dp[i][j] = 0
            else:
                if i > 0: dp[i][j] += dp[i-1][j]
                if j > 0: dp[i][j] += dp[i][j-1]
    return dp[m-1][n-1]
